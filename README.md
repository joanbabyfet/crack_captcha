## About
使用 2captcha 服务，实现破解图形验证码并自动登入博客来网站

## Feature

* 使用 2captcha 服务, 识别并绕过图形验证码
* 支持其他类型验证码, 例 reCAPTCHA V2/V3、hCaptcha、KeyCaptcha、GeeTest等

## Requires
Python 3.11.0  
selenium 4.8.0  
webdriver_manager 3.8.5   
configparser 5.3.0  

## Usage
```
python main.py
```
